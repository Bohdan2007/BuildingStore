# Z-Build Store

Курсовий проєкт: сучасний інтернет-магазин будівельних матеріалів та інструментів.

## Технології
* **Backend:** C#, ASP.NET Core MVC
* **База даних:** PostgreSQL, Entity Framework Core
* **Frontend:** HTML5, CSS3

## Основний функціонал
* Каталог товарів з прив'язкою до категорій
* Кошик та оформлення замовлення (включає вибір номера поштомату)
* Панель адміністратора для управління каталогом та обробки замовлень
* Система відгуків та формування рейтингу товарів
* Розділення прав доступу (Користувач / Адміністратор)

## Архітектура
Проєкт побудований з використанням сучасних підходів розробки:
* Патерн MVC (Model-View-Controller)
* Repository Pattern для абстрагування роботи з базою даних
* Dependency Injection (впровадження залежностей)
* Code-First підхід для генерації бази даних через Entity Framework Core
